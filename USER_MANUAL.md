# Руководство пользователя

Этот файл описывает, как пользоваться сайтом как обычный пользователь.

## 1. Вход и регистрация

1. Откройте `http://localhost:5173`
2. Если аккаунта нет:
- перейдите на экран регистрации
- заполните имя, почту и пароль
- нажмите `Создать аккаунт`
3. Если аккаунт уже есть:
- перейдите на экран входа
- введите почту и пароль
- нажмите `Войти`

## 2. Работа с постами

После входа откроется страница `Посты`.

Что можно делать:
- `Создать` — создать новый пост
- `Изменить` — изменить существующий
- `Удалить` — удалить пост
- клик по заголовку — открыть детали поста

## 3. Поиск, фильтр и пагинация

На странице списка:
- поле `поиск по заголовку` фильтрует список по title
- выпадающий список `все статусы` фильтрует по статусу
- кнопки `Назад`/`Вперёд` переключают страницы

## 4. Панель трассировки (Режим объяснения)

Панель справа показывает, что происходит внутри приложения.

Как включать/выключать:
- `Ctrl + \``

Что смотреть:
- список событий трассы
- JSON-детали выбранного события
- фильтры (trace id, тип события, endpoint, component, payload)

## 5. Карта архитектуры

В верхней части главного экрана есть `Карта архитектуры`.

Она показывает:
- какие узлы участвуют в текущем действии
- связи между экраном, API, контроллером и таблицей БД

## 6. Типовой сценарий проверки

1. Зарегистрируйтесь
2. Создайте пост
3. Измените пост
4. Удалите пост
5. Смотрите в правой панели события и детали для каждого действия
6. Смотрите подсветку узлов на карте архитектуры

## 7. Если что-то не работает

1. Проверьте, что оба сервиса запущены (`npm run dev`)
2. Проверьте адреса:
- frontend: `http://localhost:5173`
- backend: `http://127.0.0.1:8000`
3. При ошибке обновите страницу и повторите действие
